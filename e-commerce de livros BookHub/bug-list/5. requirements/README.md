# BookHub - Projeto para Prática de QA

##  Sobre o Projeto
Este é um projeto de e-commerce de livros desenvolvido para praticar habilidades de Quality Assurance. O projeto contém bugs intencionais para serem encontrados e reportados.

##  Como Executar
1. Clone este repositório
2. Abra o arquivo `index.html` no navegador
3. Explore as funcionalidades

##  Funcionalidades para Testar
- Login de usuário
- Busca de livros
- Filtros por gênero e preço
- Adicionar ao carrinho
- Remover do carrinho
- Paginação
- Responsividade

##  Bugs Conhecidos
Consulte o arquivo `bug-list/bugs-intencionais.md` para ver a lista completa de bugs planejados.

##  Como Testar
1. Execute testes manuais
2. Documente os bugs encontrados
3. Use o template de report de bug
4. Priorize por severidade

##  Tecnologias
- HTML5
- CSS3
- JavaScript Vanilla
- LocalStorage
